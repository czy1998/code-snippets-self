# code-snippets-self

一些注释的代码片段的集合

# How To Use

在 `js、ts` 文件中输入对应前缀前缀，即可应用相应的代码片段，目前有三种代码片段

```ts
输入 page
/**
 * @title ${1: 标题}
 * @pagePath ${2: 页面地址}
 * @router ${3: 路由}
 * @description ${4: 具体介绍}
 */

输入 Com (大写的C更容易找到对应的片段)
/**
 * @title ${1: 标题}
 * @description ${2: 具体介绍}
*/

输入 func
/**
 * @description ${1: 具体介绍}
 * @param ${2: 变量}
 * @return ${3: 返回结果}
*/
```
